# Integration 5A — ML Evaluation Pipeline

Module 5 Week A integration task for AI.SPIRE Applied AI & ML Systems.

## Setup

```bash
pip install -r requirements.txt
```

## Tasks

Build a structured evaluation pipeline comparing 5 model configurations (3 real models + 2 dummy baselines) using ColumnTransformer + Pipeline + cross-validation on a train/test split, plus a final held-out test set evaluation of the selected best model. See the integration guide for detailed instructions.

## Submission

Your PR description must include:
1. CV results table (all 5 models × all metrics)
2. Final test-set metrics for the selected best model
3. Recommendation paragraph (consider business context, not just accuracy)
4. Paste your PR URL into TalentLMS → Module 5 Week A → Integration 5A to submit this assignment
